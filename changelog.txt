CHANGELOG FOR WIZZY


0.2.1 - Textures
* Implemented Texture class based on 2d images to apply to shading
* Added the 'flags' parameter to ResourceManagement::Import to easily hint settings for resource loading.

0.2.2 - Mesh importing
* Implemented mesh class, containing a VertexArray and all the data that was used to create the vertex array.-
* Implemented Model resource - a list of meshes imported from model files
* Added the RenderMode option when submitting drawdata to be able to draw in line mode instead of triangle mode
